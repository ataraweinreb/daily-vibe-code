Fashion Landing Page
Prompt:
Create a responsive landing page for ___ using HTML, CSS, and JS. Use a ___ color palette. The page should feature a hero section with a stylish headline, a featured collection or product, customer testimonials, and a clear call-to-action button. Incorporate high-quality fashion imagery and modern, accessible design.
Example: Create a responsive landing page for ModeChic Boutique using HTML, CSS, and JS. Use a blush pink and charcoal color palette. The page should feature a hero section with a stylish headline, a featured summer collection, customer testimonials, and a "Shop Now" button. Incorporate high-quality fashion imagery and modern, accessible design.
Fashion Contact Page
Prompt:
Create a responsive contact page for ___ using HTML, CSS, and JS. Use a ___ color palette. Include a contact form (name, email, message), a map showing the store location, and links to social media. Style the form with modern, user-friendly design and add a section for business hours or a welcoming message.
Example: Create a responsive contact page for ModeChic Boutique using HTML, CSS, and JS. Use a blush pink and charcoal color palette. Include a contact form (name, email, message), a map showing the store location, and links to Instagram and Pinterest. Style the form with modern, user-friendly design and add a section for business hours.
Fashion Services Page
Prompt:
Create a responsive services page for ___ using HTML, CSS, and JS. Use a ___ color palette. List at least three services (such as personal styling, tailoring, wardrobe consulting), each with an icon or image, a short description, and pricing. Use a visually appealing layout, interactive hover effects, and highlight any special offers or packages.
Example: Create a responsive services page for ModeChic Boutique using HTML, CSS, and JS. Use a blush pink and charcoal color palette. List at least three services (personal styling, tailoring, wardrobe consulting), each with an icon, a short description, and pricing. Use a card layout with interactive hover effects and highlight a "Spring Refresh" package.
