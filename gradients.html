Linear Gradient Background Prompt
Prompt:
Create a responsive section with a linear gradient background using HTML, CSS, and JS. Use a ___ color gradient, ___ angle, and ___ overlay effect. The section should include a headline and a call-to-action button.
Example: Create a responsive section with a linear gradient background using HTML, CSS, and JS. Use a blue-to-teal color gradient, 135-degree angle, and a semi-transparent white overlay. The section should include a headline and a "Get Started" button.
Radial Gradient Card Prompt
Prompt:
Design a card component with a radial gradient background using HTML, CSS, and JS. Use ___ colors, ___ card size, and ___ border radius. The card should display an icon, a title, and a short description.
Example: Design a card component with a radial gradient background using HTML, CSS, and JS. Use pink and orange colors, 300x200px card size, and 20px border radius. The card should display a star icon, a title, and a short description.
Animated Gradient Text Prompt
Prompt:
Create a heading with animated gradient text using HTML, CSS, and JS. Use a ___ color gradient, ___ animation speed, and ___ font style. The text should smoothly transition through the gradient colors.
Example: Create a heading with animated gradient text using HTML, CSS, and JS. Use a purple-to-cyan color gradient, 2s animation speed, and bold, sans-serif font style. The text should smoothly transition through the gradient colors.
