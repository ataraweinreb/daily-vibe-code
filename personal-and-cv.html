Personal & CV Contact Page
Prompt:
Create a responsive contact page for ___ using HTML, CSS, and JS. Use a ___ color palette. Include a contact form (name, email, message), a map showing your location (optional), and links to social media or LinkedIn. Style the form with modern, user-friendly design and add a section for availability or a welcoming message.
Example: Create a responsive contact page for Alex Rivera Portfolio using HTML, CSS, and JS. Use a navy and coral color palette. Include a contact form (name, email, message), a map showing your location, and links to LinkedIn and Twitter. Style the form with modern, user-friendly design and add a section for availability.
Personal & CV Services Page
Prompt:
Create a responsive services page for ___ using HTML, CSS, and JS. Use a ___ color palette. List at least three services (such as freelance work, consulting, speaking), each with an icon or image, a short description, and pricing if applicable. Use a visually appealing layout, interactive hover effects, and highlight any special offers or packages.
Example: Create a responsive services page for Alex Rivera Portfolio using HTML, CSS, and JS. Use a navy and coral color palette. List at least three services (freelance web design, consulting, speaking), each with an icon, a short description, and pricing. Use a card layout with interactive hover effects and highlight a "Portfolio Review" package.
