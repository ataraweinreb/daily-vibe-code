Non-Profit Landing Page
Prompt:
Create a responsive landing page for ___ using HTML, CSS, and JS. Use a ___ color palette. The page should feature a hero section with an inspiring headline, a featured cause or project, supporter testimonials, and a clear call-to-action button. Incorporate high-quality non-profit imagery and modern, accessible design.
Example: Create a responsive landing page for GreenHope Foundation using HTML, CSS, and JS. Use a forest green and white color palette. The page should feature a hero section with an inspiring headline, a featured tree-planting project, supporter testimonials, and a "Donate Now" button. Incorporate high-quality non-profit imagery and modern, accessible design.
Non-Profit Contact Page
Prompt:
Create a responsive contact page for ___ using HTML, CSS, and JS. Use a ___ color palette. Include a contact form (name, email, message), a map showing the office location, and links to social media. Style the form with modern, user-friendly design and add a section for business hours or a welcoming message.
Example: Create a responsive contact page for GreenHope Foundation using HTML, CSS, and JS. Use a forest green and white color palette. Include a contact form (name, email, message), a map showing the office location, and links to Instagram and Facebook. Style the form with modern, user-friendly design and add a section for business hours.
Non-Profit Services Page
Prompt:
Create a responsive services page for ___ using HTML, CSS, and JS. Use a ___ color palette. List at least three services (such as volunteering, fundraising, community outreach), each with an icon or image, a short description, and pricing if applicable. Use a visually appealing layout, interactive hover effects, and highlight any special campaigns or events.
Example: Create a responsive services page for GreenHope Foundation using HTML, CSS, and JS. Use a forest green and white color palette. List at least three services (volunteering, fundraising, community outreach), each with an icon, a short description, and pricing if applicable. Use a card layout with interactive hover effects and highlight a "Spring Cleanup" campaign.
