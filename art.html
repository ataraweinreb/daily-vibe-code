Art Landing Page
Prompt:
Create a responsive landing page for ___ using HTML, CSS, and JS. Use a ___ color palette. The page should feature a hero section with an artistic headline, a featured artwork or collection, artist statements, and a clear call-to-action button. Incorporate high-quality art imagery and modern, accessible design.
Example: Create a responsive landing page for ColorMuse Gallery using HTML, CSS, and JS. Use a white and pastel color palette. The page should feature a hero section with an artistic headline, a featured abstract collection, artist statements, and a "View Collection" button. Incorporate high-quality art imagery and modern, accessible design.
Art Contact Page
Prompt:
Create a responsive contact page for ___ using HTML, CSS, and JS. Use a ___ color palette. Include a contact form (name, email, message), a map showing the gallery location, and links to social media. Style the form with modern, user-friendly design and add a section for business hours or a welcoming message.
Example: Create a responsive contact page for ColorMuse Gallery using HTML, CSS, and JS. Use a white and pastel color palette. Include a contact form (name, email, message), a map showing the gallery location, and links to Instagram and Facebook. Style the form with modern, user-friendly design and add a section for business hours.
Art Services Page
Prompt:
Create a responsive services page for ___ using HTML, CSS, and JS. Use a ___ color palette. List at least three services (such as exhibitions, art classes, commissions), each with an icon or image, a short description, and pricing. Use a visually appealing layout, interactive hover effects, and highlight any special offers or packages.
Example: Create a responsive services page for ColorMuse Gallery using HTML, CSS, and JS. Use a white and pastel color palette. List at least three services (exhibitions, art classes, commissions), each with an icon, a short description, and pricing. Use a card layout with interactive hover effects and highlight a "Summer Art Camp" package.
