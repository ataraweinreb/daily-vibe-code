Simple Footer Prompt
Prompt:
Create a simple footer using HTML, CSS, and JS. Use ___ background color, ___ text alignment, and ___ copyright style. The footer should be responsive and minimal.
Example: Create a simple footer using HTML, CSS, and JS. Use dark background color, center text alignment, and small copyright style. The footer should be responsive and minimal.
Footer with Social Links Prompt
Prompt:
Create a footer with social media links using HTML, CSS, and JS. Use ___ icon style, ___ hover effect, and ___ link arrangement. The footer should be accessible and visually appealing.
Example: Create a footer with social media links using HTML, CSS, and JS. Use outlined icon style, color-fill hover effect, and horizontal link arrangement. The footer should be accessible and visually appealing.
Footer with Newsletter Signup Prompt
Prompt:
Create a footer with a newsletter signup form using HTML, CSS, and JS. Use ___ input style, ___ button design, and ___ confirmation message. The footer should be responsive and encourage signups.
Example: Create a footer with a newsletter signup form using HTML, CSS, and JS. Use underlined input style, rounded button design, and a toast confirmation message. The footer should be responsive and encourage signups.
