Card Shadow Prompt
Prompt:
Create a card component with a custom shadow using HTML, CSS, and JS. Use ___ shadow color, ___ blur radius, and ___ hover effect. The shadow should enhance card depth and interactivity.
Example: Create a card component with a custom shadow using HTML, CSS, and JS. Use dark gray shadow color, 24px blur radius, and a lift-up hover effect. The shadow should enhance card depth and interactivity.
Button Shadow Prompt
Prompt:
Create a button with a shadow effect using HTML, CSS, and JS. Use ___ shadow offset, ___ color, and ___ active state. The shadow should provide visual feedback on interaction.
Example: Create a button with a shadow effect using HTML, CSS, and JS. Use 2px offset, blue shadow color, and a pressed-in active state. The shadow should provide visual feedback on interaction.
Image Shadow Prompt
Prompt:
Create an image with a creative shadow using HTML, CSS, and JS. Use ___ shadow shape, ___ color gradient, and ___ animation. The shadow should complement the image and add visual interest.
Example: Create an image with a creative shadow using HTML, CSS, and JS. Use oval shadow shape, purple-to-pink color gradient, and a slow floating animation. The shadow should complement the image and add visual interest.
