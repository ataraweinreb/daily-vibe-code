Hero Heading Prompt
Prompt:
Create a hero section heading using HTML, CSS, and JS. Use ___ font family, ___ font weight, and ___ text gradient. The heading should be responsive and visually striking.
Example: Create a hero section heading using HTML, CSS, and JS. Use Montserrat font family, extra-bold font weight, and a blue-to-green text gradient. The heading should be responsive and visually striking.
Body Text Prompt
Prompt:
Style body text for a website using HTML, CSS, and JS. Use ___ font size, ___ line height, and ___ color scheme. The text should be easy to read on all devices.
Example: Style body text for a website using HTML, CSS, and JS. Use 18px font size, 1.7 line height, and a dark-on-light color scheme. The text should be easy to read on all devices.
Blockquote Prompt
Prompt:
Create a blockquote style using HTML, CSS, and JS. Use ___ border style, ___ font style, and ___ background accent. The blockquote should stand out and be accessible.
Example: Create a blockquote style using HTML, CSS, and JS. Use a solid left border, italic font style, and a pale yellow background accent. The blockquote should stand out and be accessible.
