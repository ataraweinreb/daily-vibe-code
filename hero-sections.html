Hero with Image Prompt
Prompt:
Create a hero section with an image using HTML, CSS, and JS. Use ___ image placement, ___ overlay style, and ___ headline text. The hero should be fully responsive and visually engaging.
Example: Create a hero section with an image using HTML, CSS, and JS. Use right-aligned image placement, dark gradient overlay style, and bold headline text. The hero should be fully responsive and visually engaging.
Hero with Video Prompt
Prompt:
Create a hero section with a background video using HTML, CSS, and JS. Use ___ video source, ___ overlay color, and ___ call-to-action button. The hero should autoplay the video and be accessible.
Example: Create a hero section with a background video using HTML, CSS, and JS. Use mp4 video source, semi-transparent black overlay color, and a large call-to-action button. The hero should autoplay the video and be accessible.
Hero with Call-to-Action Prompt
Prompt:
Create a hero section with a call-to-action using HTML, CSS, and JS. Use ___ background style, ___ headline, and ___ button design. The hero should be responsive and encourage user interaction.
Example: Create a hero section with a call-to-action using HTML, CSS, and JS. Use gradient background style, inspiring headline, and pill-shaped button design. The hero should be responsive and encourage user interaction.
